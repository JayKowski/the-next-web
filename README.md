## General info
The above project was undertaken so as to delve deep into the architecture of <a href="https://thenextweb.com/">the next web</a> website and also used as a practical study case on building with responsive design. The above mentioned website is one dedicated to the dissemination of information on current developments and happenings in the tech world. From the aforementioned website, the above project was built. It is a collaborative effort by <a href="https://github.com/adaorachi">MaryAnn Chukwuka</a> and <a href="https://github.com/JayKowski"> Judah Maina </a>.
